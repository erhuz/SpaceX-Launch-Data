# SpaceX Launch Data
A React-based client that retrieves data from from the community-driven SpaceX API.


## Install
1. Clone directory
```
  git clone https://github.com/erhuz/SpaceX-Launch-Data.git
```
2. cd into directory
```
  cd SpaceX-Launch-Data
```
3. Install dependencies
```
  yarn install
```
4. Start the app
```
  yarn start
```
